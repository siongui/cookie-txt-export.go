Google Chrome extension that creates a cookie.txt file for use with wget,
written in Go_, compiled to JavaScript via GopherJS_.
(Go porting of cookie-txt-export_)

Tested on:

- `Ubuntu Linux 17.10`_
- `Go 1.9.3`_
